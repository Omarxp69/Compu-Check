💻 ¿Qué es Compucheck?

Compucheck es una aplicación web modular diseñada para administrar y controlar los equipos de cómputo. Su principal objetivo es optimizar la gestión de los salones de cómputo, permitiendo registrar, consultar y actualizar información sobre las computadoras y sus componentes (monitor, teclado, mouse) de manera centralizada, rápida y confiable.

El sistema está desarrollado con tecnologías como Python, Flask, HTML, CSS, JavaScript y MySQL, y utiliza un modelo de inteligencia artificial basado en redes neuronales convolucionales (CNN) para analizar imágenes y verificar automáticamente el estado de los periféricos.

Gracias a su diseño web, Compucheck puede ser accedido desde cualquier navegador sin necesidad de instalación, facilitando el trabajo colaborativo entre administradores, técnicos y usuarios. Además, su base de datos relacional garantiza la integridad, consistencia y trazabilidad de la información, reduciendo errores humanos y mejorando la eficiencia en la supervisión y mantenimiento de los equipos.

¿Listo para mejorar la gestión de tus salones? Comienza hoy con Compucheck y transforma el mantenimiento de tus equipos en una tarea automatizada y eficiente.

Manual de usuario

1. 🏠 Inicio

Este es la ventana principal de la pagina web

2. 📝 Regístrate

Apartado para registrarte, ingresa tus datos y recibirás un correo electrónico en spam para poder iniciar sesión

3. 🔑 Iniciar sesión

Después de realizar el registro y verificar el correo electrónico, iniciamos sesión con los datos que utilizamos para registrar.

4. 📊 Reportes

Como usuario podemos generar reportes

5. -> Selecciona una de las salas ya creadas por algún administrador para empezar a generar el reporte.

6. -> Selecciona la computadora que se encuentra en la sala para generar el reporte

7. -> Seleccionamos un archivo para capturar la pantalla

8. -> Seleccionamos un archivo para capturar el teclado

9. -> Seleccionamos un archivo para capturar el mouse

10. -> Se puede agregar comentarios sobre el estado del equipo para después darle en "Hacer Reporte"

11. ✏️ Editar Perfil

En editar perfil podemos cambiar la 🖼 Foto de perfil después de seleccionar el archivo damos a actualizar foto.

12. -> Podemos actualizar la 🎂 Fecha de nacimiento del usuario, Agregar el ⚧️Genero de el usuario y ❌Eliminar tu cuenta

13. 👥 Gestionar Usuarios - Moderador

Podemos buscar un usuario por ID, NOMBRE, APELLIDOS, EMAIL ROL, ESTADO, FECHA DE CREACION Y ACTUALIZACION

14. 🛠️ Gestionar Salas - Moderador

Podemos visualizar las salas en el sistema por medio de los siguientes filtros -> buscar una sala por ID, NOMBRE, APELLIDOS, EMAIL, UBICACION, ESTADO, FECHA DE CREACION Y ACTUALIZACION

15. 🔑 Permisos Usuarios - Admin/Moderador

Los Administradores y moderadores pueden darle a los usuarios un permiso para realizar reportes en diferentes salas

16. -> Podemos filtrar usuarios y salas

17. -> Podemos visualizar los usuarios y las salas que queremos asignar permisos

18. 🛠️ Gestionar Permisos - Admin/Moderador

Podemos Gestionar Permisos, como eliminar los permisos otorgados

19. 🛠️ Gestionar Reportes - Admin/Moderador

Podemos Gestionar Reportes, como eliminar los reportes creados

20. 🏠 Inicio - Admin

Dashboard de administradores

21. 👥 Gestionar Usuarios - Admin

Los administradores pueden Modificar los roles y activar y desactivar la cuenta de todos los usuarios en el sistema usuario

Los administradores tienen el apartado de salas donde se pueden crear salas nuevas en la base de datos

23. 🛠️ Gestionar Salas

Aqui los administradores pueden Modificar los estados de las salas y eliminar salas

24. -> Visualización de las salas existentes en la base de datos

25. 💻 Computadoras - Admin

Aqui los administradores Pueden Crear Computadoras y asignarlas a una sala

26. 🛠️ Gestionar Computadoras

En este apartado se visualizan las computadoras que hay en la base de datos y se puede modificar su estado
